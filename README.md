<h1>Project Name: NFT Marketplace Frontend</h1>

<h3>Description:</h3>
<p>This project is a frontend implementation of an NFT (Non-Fungible Token) Marketplace, developed as a part of the technical club recruitment process for the blockchain club at IIT Dhanbad. The project focuses on creating a user-friendly interface for browsing and interacting with NFTs.</p>
